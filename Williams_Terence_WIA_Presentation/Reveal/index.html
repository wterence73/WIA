AniJS

A jQuery plugin that animates elements

by Terence Williams

Let's go through it together!

Why did I choose AniJS?

Animation!

I chose aniJS because of its use for animating the web.

A small package with a LARGE presentation!

AniJS is a jQuery plugin that uses sort of the if/and statements. It is simply nested next to the element one wishes to animate.

What a reminder!

We are all so familiar with Google search. AniJS reminded me of how they animated that page so well. It catches the users attetion and brings them back not only for its effectiveness but its interactivity.

What can it animate?

  • Headers
  • Footers
  • Buttons
  • ...and many other elements

How do we use aniJS?

  1. You have to include the aniJS library.
  2. You can optionally include some css animation definitions.
  3. Choose the element you wish to animate within your code.
  4. Next to the element, you will add the data-anijs tag.
  5. Lastly, you add the desired if/and statements to get desired animation. (There are plenty!)

Example 1

Code


<!DOCTYPE html>
<html lang="EN">

    <head>
        <meta charset="utf-8">
        <title>WIA-Presentation</title>
        <link type="text/css" href="css/style.css" rel="stylesheet">
        <link rel="stylesheet" href="http://anijs.github.io/lib/anicollection/anicollection.css">
    </head>
    
    <body>
        
        <article>
            
            <header data-anijs="if: mouseover, do: swing animated"><span class="my">my</span><span class="g">S</span><span class="o">e</span><span class="O">a</span><span class="G">r</span><span class="l">c</span><span class="e">h</span></header>
            
            <form>
                <input type="text"/>
            </form>
            <button data-anijs="if: mouseover, do: hinge animated">mySearch</button>
            <button type="submit" data-anijs="if: mouseover, do: rubberBand animated">I'm Feeling Lucky</button>
        
        </article>
        
        <script src="http://anijs.github.io/lib/anijs/anijs-min.js"></script> 
  
        <script src="http://anijs.github.io/lib/anijs/helpers/dom/anijs-helper-dom-min.js"></script>
        
    </body>

    
</html>
